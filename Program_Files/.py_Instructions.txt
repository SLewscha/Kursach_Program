This program can be launched though the command line.
For this you need to follow these steps:

1) Launch the command line application.

2) Enter the command that will take you to the directiory where all three program files ("main.py", "sender.py", "receiver.py") are located.
	The command looks like this: "cd 'path to the directory'".

3) Launch the program by entering the following command: "python main.py"
	Is is important to node that for this command to work you must have a Python interpreted installed on your PC.
